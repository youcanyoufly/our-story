# 我们的故事 - 项目记录

## 项目简介
为妻子生日制作的胶片电影风格网页，展示两人的爱情故事。

## 技术栈
- 纯 HTML + CSS + JavaScript（单文件方案）
- GitHub Pages 部署
- GitHub API 实现在线管理

## 项目结构
```
our-story/
├── index.html        ← 主页面（胶片电影风格）
├── admin.html        ← 管理后台（在线编辑/上传照片）
├── config.js         ← 配置文件（所有文字内容）
├── CLAUD.md          ← 本文件
└── photos/
    ├── act1/         ← 第一幕：相遇
    ├── act2/         ← 第二幕：相知
    ├── act3/         ← 第三幕：相守
    └── special/      ← 片尾特别照片
```

## 功能特性

### 视觉效果
- 老式电影倒计时开场（3、2、1）
- 胶片颗粒纹理 + 暗角 + 扫描线
- 信封/信纸背景（泛黄纸张、手写横线、左侧红线、火漆印章）
- 胶片条横向滚动（鼠标悬停暂停）
- 照片黑白→彩色渐变效果
- 打字机逐字打出情话
- 电影片尾字幕滚动
- 花瓣飘落动画

### 交互功能
- 双击照片放大查看（灯箱）
- 多张照片左右切换（键盘←→或点击箭头）
- ESC 或点击空白处关闭灯箱
- 背景音乐控制（放 bgm.mp3 到目录即可）
- 滚动时元素淡入动画

### 管理后台（admin.html）
- 在线编辑所有文字内容
- 上传照片（自动转为 jpg，兼容所有手机格式）
- 删除照片
- 一键发布到 GitHub
- Token 保存在浏览器本地

## 关键配置

### config.js 编辑项
- `hisName` / `herName`：两个人的名字
- `togetherDate`：在一起的日期
- `act1/act2/act3`：每一幕的标题、日期、故事、照片
- `ending`：片尾照片、祝福语、字幕

### 照片要求
- 格式：统一为 jpg（管理后台自动转换）
- 胶片条照片：建议横拍，比例接近 3:2
- 大照片（act2/special）：建议横拍，比例接近 3:2
- 照片数量：胶片条可放任意张，最少 1 张
